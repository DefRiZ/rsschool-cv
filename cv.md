**Code Example**

**Vowel remover:** _Create a function called shortcut to remove the lowercase vowels (a, e, i, o, u ) in a given string._

```javascript
function shortcut(string) {
  const notAllowed = ["a", "e", "i", "o", "u"];
  return string
    .split("")
    .filter((el) => !notAllowed.includes(el))
    .join("");
}
```
